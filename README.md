# Portfolio Desain Dapur

Website portfolio arsitektur khusus desain dapur dengan video hero embedded dari YouTube.

## Cara Menjalankan
Buka `index.html` di browser:
- Windows: `start index.html`

Video akan autoplay (muted) di halaman utama. Responsive untuk mobile.

## Struktur
- `index.html`: Halaman utama (Bahasa Indonesia)
- `css/styles.css`: Styling modern, tema kayu/coklat
- `js/script.js`: Interaksi (menu mobile, scroll, animasi)
- Gambar: Unsplash kitchen designs

Edit konten proyek sesuai kebutuhan!
